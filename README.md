# EV Charge and Range Analysis Dashboard

## Project Overview
This project is an interactive Tableau dashboard that analyzes Electric Vehicle (EV) charging and range data. It provides insights into vehicle performance, charging capabilities, battery efficiency, and brand-wise comparisons using multiple datasets.

## Objectives
- Analyze EV charging and range performance.
- Compare different EV brands.
- Visualize battery capacity, top speed, efficiency, and powertrain.
- Provide interactive filters for better data exploration.

## Tools Used
- Tableau Desktop
- Tableau Public
- MySQL
- GitHub

## Datasets Used
This project uses four datasets containing:
- Cheapestelectriccars-EVDatabase
- electric_vehicle_charging_station_list
- ElectricCarData_Clean
- EVIndia

## Dashboard Features
- Brand-wise Analysis
- Powertrain Analysis
- Top Speed Comparison
- Battery Capacity Analysis
- Efficiency Analysis
- Interactive Filters
- Charts including Bar Charts, Pie Charts, and Line Charts
